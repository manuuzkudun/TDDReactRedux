#DDReactRedux#
A simple ####React-Redux#### app to post comments developed using Test Driven Development.
Testing was done using ####mocha#### and ####chai####.

####installation Instructions####
```
	> git clone https://github.com/manuuzkudun/TDDReactRedux.git
	> cd TDDReactRedux
	> npm install
	> npm start
```
